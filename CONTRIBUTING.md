# Contributing to Women Safety App

## How to Contribute

We welcome contributions! Here's how you can help:

### Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/women-safety-app.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Push to your fork: `git push origin feature/your-feature-name`
6. Create a Pull Request

### Development Setup

#### Backend
```bash
cd backend
npm install
npm run dev
```

#### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Code Standards

- **Frontend**: Run `npm run lint` before committing
- **Backend**: Keep code clean and well-documented
- **Commits**: Use clear, descriptive commit messages
- **Tests**: Add tests for new features when applicable

### Pull Request Process

1. Update relevant documentation
2. Ensure your code passes linting and builds
3. Reference any related issues in the PR description
4. Wait for code review and address feedback

### Code of Conduct

- Be respectful and inclusive
- No harassment or discrimination
- Focus on constructive feedback

Thank you for contributing to making women safer! 🙏
